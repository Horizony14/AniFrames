## 简介

这里是二次元游戏（二游）相框网站 **AniFrameLab** 的相框仓库，集中存放网站所使用的相框图片素材。

- 🌐 网站地址：<https://aniframelab.pages.dev/>
- 🖼️ 仓库内容：各二游的相框素材（`.webp` 格式）
- 🎨 站点图标：`icon_n.webp`（浅色 / 日间）与 `icon_d.webp`（深色 / 夜间），即上方展示的两个图标

## 关于素材

- 图片原素材均来自对应游戏，版权归各游戏官方所有。
- 本仓库仅对原素材进行了 **抠图** 与 **二次加工**（裁切、抠除背景、格式转换等），方便在网站中作为相框使用。
- 本仓库不持有任何素材的原始版权，相关权利归属各游戏厂商。

## 使用说明

- 所有素材仅供个人学习、交流与非商业用途。
- 如需在自己的项目中使用，请遵循对应游戏官方的素材使用规范，并保留来源说明。
- 若有侵权问题，请通过 issue 联系，会及时处理。
